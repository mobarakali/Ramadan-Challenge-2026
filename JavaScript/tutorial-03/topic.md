# JavaScript Tutorial 03 - JavaScript Comments

Tutorial Link : https://www.youtube.com/watch?v=BVo3rTdAyjo

## Topics Covered

### Singel Line Comment

```javascript
    // window.alert("JavaScript Alert!");
    // Above is an example of a Single line Comment
```

### Block Comment or Multiline Line Comment

```javascript
      /* window.*/ alert("JavaScript Alert!");
      // Above is example of Block Comment

      /* 'Block Comment' is oftern reffered as 'Multiline Comment'
      because one can comment several lines of code or text using 
      'Block Comment' or 'Multiline Comment' like this one.*/
```

Using comment we can prevent code execution of that line or block, additionaly we can provide instruction or information
related to that code.
