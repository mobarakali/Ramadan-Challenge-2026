# JavaScript Tutorial 05 - JavaScript Variable

## Topics Covered

### Why Variables are important

ভেরিয়েবল ছাড়া আধুনিক প্রোগ্রামিং কল্পনাই করা যায় না। এর গুরুত্ব নিচের কারণগুলো থেকে বোঝা যায়:

- **ডাটা সংরক্ষণ করা যায়**  
  প্রোগ্রাম চালানোর সময় আমাদের বিভিন্ন তথ্য (নাম, বয়স, স্কোর, দাম, টেক্সট ইত্যাদি) কম্পিউটারের মেমরিতে রাখতে হয়। ভেরিয়েবল হচ্ছে সেই **ধারক/বাক্স/পাত্র**, যেখানে আমরা এই ডাটা রাখি।

- **ডাটা পুনরায় ব্যবহার করা সহজ হয়**  
  একবার ভেরিয়েবলে মান রাখলে পরে শত শত জায়গায় সেই মান ব্যবহার করা যায় শুধু ভেরিয়েবলের নাম লিখে। যেমন:  
  ```js
  let totalPrice = 1500;
  console.log("মোট দাম: " + totalPrice);
  console.log("ডিসকাউন্ট পরে: " + (totalPrice - 200));
  ```

- **কোড পরিবর্তন করা সহজ হয় (Maintainable)**  
  ধরো ১০০ জায়গায় একই দাম লিখে রেখেছ — পরে দাম বদলাতে হলে ১০০ জায়গা বদলাতে হবে। কিন্তু ভেরিয়েবল ব্যবহার করলে শুধু এক জায়গায় বদলালেই হয়।

- **প্রোগ্রামকে গতিশীল (Dynamic) করে**  
  ব্যবহারকারীর ইনপুট, API থেকে আসা ডাটা, গণনার ফলাফল — সবকিছু ভেরিয়েবলে রেখে আমরা প্রোগ্রামকে জীবন্ত করে তুলি।

- **কোড পড়তে ও বুঝতে সুবিধা হয়**  
  `let userAge = 25;` লিখলে সাথে সাথে বোঝা যায় এটা বয়স। কিন্তু সরাসরি 25 লিখে রাখলে পরে বোঝা কঠিন হয়ে যায়।

সংক্ষেপে: **ভেরিয়েবল না থাকলে প্রোগ্রামিং হতো খুব জটিল, অগোছালো এবং পরিবর্তন করা প্রায় অসম্ভব।** এটি প্রোগ্রামের ভিত্তি।

### What is a Variable

**সহজ ভাষায়:**  
ভেরিয়েবল হচ্ছে **একটা নামকরা বাক্স** বা **ধারক**, যার ভেতরে আমরা যেকোনো মান (value) রাখতে পারি। এই মান পরে পরিবর্তনও করা যায় (যার জন্যই নাম "variable" — পরিবর্তনশীল)।

**উদাহরণ দিয়ে বোঝা যাক:**

```js
let age;          // একটা খালি বাক্স তৈরি করলাম, নাম দিলাম age
age = 10;         // বাক্সের ভেতর ১০ রাখলাম

console.log(age); // বাক্স থেকে মান বের করে দেখাল → 10

age = 15;         // একই বাক্সে এখন ১৫ রাখলাম (মান পরিবর্তন হলো)
console.log(age); // এখন → 15
```

**আরেকটা বাস্তব উদাহরণ:**

```js
let productName = "মোবাইল ফোন";
let price = 25000;
let discount = 10;

let finalPrice = price - (price * discount / 100);

console.log(productName + " এর দাম ছিল " + price + " টাকা।");
console.log("ডিসকাউন্ট পরে দাম: " + finalPrice + " টাকা।");
```

**সারাংশে ভেরিয়েবল মানে:**
- একটা **নাম** (যেমন: age, price, name)
- যার ভেতরে **ডাটা/মান** রাখা যায় (number, text, true/false ইত্যাদি)
- যাকে **পরে ব্যবহার** করা যায়, **পরিবর্তন** করা যায়

তোমার দেওয়া কোডে দেখেছ যে JavaScript-এ ভেরিয়েবল তৈরি করার তিনটা প্রধান উপায় আছে:  
`var` (পুরোনো), `let` (আধুনিক, ব্লক-স্কোপড), `const` (যার মান আর বদলানো যাবে না)

**শুরুতে সাজেশন:** নতুন কোড লিখতে গেলে **let** আর **const** ব্যবহার করো। `var` এখন প্রায় আর ব্যবহার করা উচিত না (শুধু খুব পুরোনো কোডে থাকতে পারে)।
